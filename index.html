What is a Claude Skill?

A Claude Skill is a folder containing instructions, scripts, and resources that Claude loads dynamically when needed. Skills teach Claude how to complete specific tasks in a repeatable way, improving consistency and performance.

The Minimal Design System Skill gives Claude precise instructions on how to create professional, minimalist websites and components with consistent styling, generous whitespace, and clear hierarchy.

Claude automatically invokes Skills based on your taskâ€”no manual selection needed. When you describe a task that matches a Skill's description, Claude recognizes it and loads the Skill automatically.

How to Use This Skill

In Claude.ai (Web & Desktop App)

1

Go to Settings â†’ Capabilities and ensure "Skills" is enabled. (Note: Skills require Code execution to be enabled)

2

Click the "Download Complete Skill" button in the header above. This downloads a ready-to-upload ZIP file with all templates and references.

3

Upload to Claude

In Claude, go to Settings â†’ Capabilities â†’ Skills

Click "Upload skill" and select the ZIP file

4

Use the Skill

Simply describe your task in natural language. Claude will automatically recognize when to use the Skill:

"Create a landing page for my product" "Build a minimal dashboard" "Use minimal design system for this website"

In Claude Code (Terminal)

1

Restart Claude Code

Restart Claude Code to load the new Skill. It will be automatically available in all your sessions.

ChatGPT supports custom instructions and file uploads:

  4. 4. Upload additional files from assets/ and references/ as needed during your conversation
